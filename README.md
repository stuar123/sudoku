## 查看方式

### GitHub

访问 [目录索引](SUMMARY.md)

### 脑图

访问 [数独资源与分类](http://naotu.baidu.com/file/825e8b6bb24556bf904887a36158afcd?token=2739364acfe6d714)

###Gitbook

1. clone 代码，并启动 gitbook
    ```shell
    git clone git@github.com:zhugelianglongming/sudoku.git
    cd sudoku
    gitbook serve
    ```
2. 访问 http://localhost:4000 查看
